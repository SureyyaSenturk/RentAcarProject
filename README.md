# CAR RENTAL PROJECT
+ Admin can see the bookings and can manage the cars. <br/>
+ If a car is booked for a time span no other customer can book the same car on same time span.
## Technologies Used <br/>
 ### Backend
 - SpringBoot <br/>
 - Spring-Security
 - ModelMapper
 - Postgresql-Database
 - Spring-Jpa
 - OpenApi-Swagger
 - Spring-Validation
 
 ### Front-End
 - React <br/>
 - TypeScript
 - Tailwind Css
 - Axios
